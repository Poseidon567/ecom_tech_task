Репозиторий для выполненного тестового задания: 
Разработать REST-сервис на FastAPI для загрузки и анализа успеваемости студентов.
Сервис должен принимать .csv файл, сохранять данные в БД и предоставлять ручки для анализа.
Сервис должен:
1. Принимать CSV-файл и сохранять данные в PostgreSQL.
2. Выполнять валидацию данных.
3. Предоставлять две аналитические ручки.
4. Работать строго через SQL-запросы — использование ORM запрещено.
5. Возвращать данные в формате JSON.

Ожидаемый результат:
Ссылка на GitHub репозиторий, содержащий:
– исходный код FastAPI;
– инструкцию по запуску;
– SQL-скрипт создания таблиц или миграции.

Будет плюсом:
– Написать тесты
– Упаковать в docker (compose)
– Написать миграции для БД

Инструкция по запуску находится в файле Instructions.txt
SQL-скрипт создания таблиц или миграции находится в файле create_tables.sql
